# refactoring 2nd training

## 概要

[リファクタリング第二版](https://www.amazon.co.jp/dp/B0827R4BDW) の練習用です。

ES2017 & jest が使えるようにしてあります。

## インスコ

```sh
% git clone https://github.com/Tommy1969/refactoring2nd.git
% cd refactoring2nd
% yarn install
```

## 使い方

実行
```sh
% yarn start
```

テスト
```sh
% yarn test
```

テスト:ウォッチモード
```sh
% yarn test:watch
```

Enjoy hacking! :-)
